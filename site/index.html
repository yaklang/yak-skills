Install

把 Yak Skills 接入你的 Agent：整包安装，或按需拉取单个专题。

一条命令拉取整套（总入口 + 5 个专题 skill）并注册到你的 skills loader。

$ npx skills add yaklang/yak-skills

首次接入推荐。兼容任何遵循 skills CLI 约定的工具。

只装一个专题——当你只需要某一类能力、想保持 loader 精简时使用。

$ npx skills add yaklang/yak-skills/yak

若工具支持按 skill 注册（frontmatter name 标识），只会引入这一个专题。

纯 HTTP。拉取任意 skill 的原始 SKILL.md，丢进自定义 prompt、RAG 或自己的 skills 目录。

$ curl -fsSL https://raw.githubusercontent.com/yaklang/yak-skills/main/skills/yak/SKILL.md

无需安装器。Markdown 自包含，frontmatter 内含路由提示。

把一个 URL 交给你的 AI Agent —— Cursor、Claude Desktop、MCP 风格 loader 或任意读取远端 Markdown 的工具。

frontmatter name yak
raw url https://raw.githubusercontent.com/yaklang/yak-skills/main/skills/yak/SKILL.md
github source github.com/…

多数 Agent loader 会把 frontmatter 的 name: 当作稳定标识，即使源 URL 变化。
